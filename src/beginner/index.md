# JavaScript Beginner

[Hub](https://alibaihaqi.github.io/learning-docs/) › JavaScript › Beginner

## What you'll build

A command-line todo app in plain JavaScript on Node.js — no frameworks, no
third-party packages. `node cli.js add "buy milk"` appends a todo to a
`todos.json` file; `node cli.js list` prints your todos as numbered rows. The
todo logic lives in a small module you test with Node's built-in test runner.
