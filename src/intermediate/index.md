# JavaScript Intermediate

[Hub](https://alibaihaqi.github.io/learning-docs/) › JavaScript › Intermediate

## What you'll build

You grow the beginner CLI todo into a fuller tool: mark todos done, remove them,
filter to just the pending ones, and seed the list asynchronously from a public
API with `fetch` and `async/await`. Still plain Node, still zero dependencies.
